**o***rganize***date** is lightweight pkg that converts timestamps (or `Date`) into clean, human-readable dates — including “seconds ago”, “Yesterday at 14:23”, or full formatted dates with optional seconds.

## Install
```shell
npm i @arisyntek/odate
```

## Use
```typescript
import { formatDate } from '@arisyntek/odate';

const date = formatDate(1763242535431);
```
